### Key Features
- Responsive design

- Interactive product cards
- Social media integration

### UI Components
- **Navigation Bar**
  - Responsive menu

  - Smooth transitions

- **Hero Section**
  - Full-width banner
  - Call-to-action buttons
  - Optimized background images

- **Product Cards**
  - Hover effects
  - Price display
  - Add to cart functionality
  - Image galleries

- **Footer**

  - Contact information
  - Navigation links

- Next.js Image optimization
- Lazy loading

- Efficient routing
